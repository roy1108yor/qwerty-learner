# Qwerty Learner 项目分析报告

## 项目概述

Qwerty Learner 是一款专为键盘工作者设计的英语单词学习工具，旨在提高用户的英语输入效率和准确度。该项目通过结合打字练习和单词记忆，创造了一种高效、实用的学习方式，使用户在提高打字速度的同时能够有效地学习和记忆英语单词。

核心功能包括：
- 单词打字练习
- 多种词库选择
- 发音功能
- 记录错误单词
- 数据统计与分析
- 单词默写模式
- 打字速度和准确率统计

## 技术栈分析

Qwerty Learner 采用现代前端技术栈构建：

- **前端框架**：React 18
- **开发语言**：TypeScript
- **样式解决方案**：Tailwind CSS + CSS Modules
- **路由管理**：React Router
- **状态管理**：Jotai + React Context + useImmer
- **构建工具**：Vite
- **UI组件**：
  - Radix UI (提供无样式的可访问性组件)
  - Headless UI
  - Floating UI
  - Lucide React (图标库)
- **图表可视化**：ECharts
- **本地存储**：Dexie.js (IndexedDB 封装)
- **音频处理**：Howler.js + use-sound
- **开发工具**：ESLint + Prettier
- **跨平台支持**：Web平台 + Tauri(桌面应用)

## 核心功能模块分析

### 打字练习功能（Typing模块）

Typing 模块是应用的核心功能，提供单词打字练习体验：

- **单词展示与打字**：展示当前单词及翻译，用户通过键盘输入对应内容
- **单词发音**：支持自动或手动播放单词发音
- **即时反馈**：字母输入正确/错误的视觉反馈
- **打字统计**：实时显示速度(WPM)和准确率
- **默写模式**：只展示翻译不展示单词，用户需凭记忆输入
- **跳过功能**：允许跳过当前难词
- **键盘音效**：支持多种键盘按键音效
- **结果页面**：完成练习后展示统计数据和错误单词

该模块使用 React Context 结合 useImmerReducer 管理复杂状态，处理用户输入、计时、单词切换等逻辑。

### 词库管理与单词展示功能（Gallery模块）

Gallery模块提供词库浏览和选择功能：

- **多语言支持**：支持英语、德语、日语等多种语言
- **分类展示**：根据类别（如四六级、托福、编程等）展示词典
- **标签过滤**：通过标签快速筛选词典
- **词库详情**：展示词库的单词数量、难度等信息
- **章节管理**：将大型词库分为多个章节便于学习
- **学习记录**：显示用户对每个章节的学习进度

该模块使用 useImmer 管理状态，实现了高效的词库筛选和展示功能。

### 错误单词管理功能（ErrorBook模块）

ErrorBook模块记录用户在练习中出错的单词：

- **错误单词列表**：展示用户输入错误的单词
- **错误次数统计**：记录每个单词被错误输入的次数
- **排序功能**：支持按错误次数排序
- **单词详情**：点击单词可查看详细错误记录
- **删除功能**：支持删除不需要关注的错误记录
- **导出功能**：支持导出错误单词列表
- **分页显示**：分页展示大量错误记录

该模块通过 IndexedDB (Dexie.js) 持久化存储错误记录，确保数据不会丢失。

### 数据分析功能（Analysis模块）

Analysis模块提供用户学习数据的可视化分析：

- **练习热力图**：展示过去一年的练习频率
- **单词量热力图**：展示过去一年练习的单词数量
- **WPM趋势图**：展示打字速度的变化趋势
- **正确率趋势图**：展示正确率的变化趋势
- **错误键位分析**：展示键盘上最容易出错的按键

该模块使用 ECharts 实现数据可视化，帮助用户了解自己的学习状况和进步。

## 数据流分析

Qwerty Learner 采用多种状态管理解决方案：

1. **Jotai**：用于跨组件共享的全局状态
   - 当前词典、章节信息
   - 用户配置（如暗黑模式、发音设置等）
   - 复习模式信息

2. **React Context + useImmerReducer**：用于复杂组件内部状态
   - 打字练习状态（当前单词、输入内容、计时等）
   - Gallery页面状态（当前语言标签等）

3. **本地存储**：
   - **IndexedDB (Dexie.js)**：存储用户练习记录、错误单词数据
   - **localStorage**：存储用户配置信息

数据流向：
- 用户交互 → 状态更新 → UI渲染
- 打字记录 → 本地数据库存储 → 数据分析展示

## 主要页面组件结构

1. **Typing页面**：
   - WordPanel：单词显示和输入处理
   - Speed：速度和准确率显示
   - PronunciationSwitcher：发音控制
   - StartButton：开始/重置练习
   - ResultScreen：练习结果展示

2. **Gallery页面**：
   - LanguageTabSwitcher：语言切换
   - CategoryDicts：词典分类展示
   - Dictionary：词典卡片组件
   - ChapterGroup：章节管理

3. **ErrorBook页面**：
   - ErrorRow：错误单词行
   - Pagination：分页控制
   - RowDetail：单词错误详情
   - HeadWrongNumber：排序控制

4. **Analysis页面**：
   - HeatmapCharts：热力图
   - LineCharts：趋势线图
   - KeyboardWithBarCharts：键盘错误分析

## 核心功能亮点

1. **单词发音功能**：
   - 支持自动/手动播放发音
   - 使用 Howler.js 处理音频

2. **默写模式**：
   - 隐藏单词，仅显示翻译
   - 强化记忆训练

3. **速度和准确率统计**：
   - 实时计算WPM（每分钟字数）
   - 记录并展示正确率

4. **数据可视化**：
   - 多种图表展示学习情况
   - 帮助用户了解学习趋势

5. **多语言支持**：
   - 支持多种语言词库
   - 适应不同语言学习需求

6. **个性化配置**：
   - 键盘音效选择
   - 暗黑/明亮模式切换
   - 打字显示设置

7. **离线存储**：
   - IndexedDB存储学习记录
   - 支持数据导入导出

Qwerty Learner 通过将打字练习与语言学习相结合，创造了一种高效、有趣的学习方式，特别适合需要同时提高英语水平和打字能力的用户。
